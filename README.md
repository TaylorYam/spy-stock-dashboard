# spy-stock-dashboard — S&P 500 個股動能 Streamlit dashboard

互動式 dashboard，讓使用者試算與視覺化 S&P 500 個股動能策略。

## Stack
- Streamlit
- Python: pandas / numpy
- Data：後端 [QuantCore](https://github.com/TaylorYam/QuantCore) 產出

## 配套
- [sm](https://github.com/TaylorYam/sm) — 同策略的每日網頁週報（純 HTML）
- [sr](https://github.com/TaylorYam/sr) — 姊妹版產業 ETF 輪動週報
- [TaylorYam.github.io](https://tayloryam.github.io/) — 個人首頁
